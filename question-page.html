Question-1: What are the different ways to select an element in the DOM?

Answer: There are many ways to select an element in DOM. Some of these are:
1.getElementById()
2.getElementsByClassName()
3.getElementsByTagName()
4.querySelector()
5.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML: It sets HTML content, including tags.
innerText: It sets only the visible text, considering CSS styles.
textContent: It is raw text, unaffected by CSS styles and includes hidden text.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a powerful technique for handling events more efficiently in JavaScript by using the event bubble mechanism.
Instead of attaching individual event listeners to each child element, it attaches one event listener to a parent element and use event.target to identify which child triggered the event.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling means that an event triggered on an element will propagate or bubble up to its parent elements.It helps to simplify handling events,
especially in cases where you have a lot of elements or dynamic content.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:We can use some functions to create, add and remove elements using JavaScript. These are,
1.We can use document.createElement() to create an element using JavaScript.
2.We can use appendChild() to add an element to he webpage using JavaScript.
3.We can use remove() to remove an element from the webpage using JavaScript.
